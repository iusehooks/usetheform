# useMultipleForm
`useMultipleForm(callback: Function)` provides the logic to handle the implementation of wizards.

```javascript
  const [getWizardStatus, wizardApi] = useMultipleForm(callback);
```

## Returns

(**`helpers`**): array

An array that holds:

  - **`getWizardStatus`**: It is a function that returns the actual state of the wizard..
  - **`wizardApi`**: an object which holds props that must be spreaded in each form.

## Basic usage

```javascript
  import { Form, Input, useMultipleForm } from 'usetheform'
```

<Playground>
{() => {
    const [getWizardStatus, wizardApi] = useMultipleForm();
    const [stateWizard, setStateWizard] = useState({})
    return (
      <div>
        <Form {...wizardApi} name="form1" initialState={{ name: "foo" }}>
          <Input type="text" name="name" />
        </Form>
        <Form {...wizardApi} name="form2" initialState={{ lastname: "mouse" }}>
          <Input type="text" name="lastname" />
        </Form>

        <div>
          <code>{JSON.stringify(stateWizard)}</code>
        </div>
      </div>
    )
  }
}
